# Java Calculator

A simple menu-driven calculator built in Java to practice core programming concepts such as user input, methods, loops, conditional statements, and switch-case operations.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Power Calculation
* Input Validation
* Divide-by-Zero Handling
* Repeat Calculations Using a Do-While Loop

## Concepts Used

* Java Fundamentals
* Scanner Class
* Methods and Return Values
* Switch Statements
* Loops (Do-While)
* Conditional Logic
* Math.pow()

## Sample Run

```
===== CALCULATOR =====

Enter first number: 10
Enter second number: 5

Choose Operation
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
5. Modulus (%)
6. Power (^)

Enter choice: 3

Result = 50.0

Do another calculation? (y/n): y
```
