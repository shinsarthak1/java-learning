# Temperature Converter

A menu-driven Java console application that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Features

- Celsius → Fahrenheit
- Fahrenheit → Celsius
- Celsius → Kelvin
- Kelvin → Celsius
- Fahrenheit → Kelvin
- Kelvin → Fahrenheit
- Input validation
- Loop for multiple conversions
- Formatted output

## Technologies

- Java
- Scanner
- Switch Statements
- Methods

## How to Run

Compile:

javac TemperatureConverter.java

Run:

java TemperatureConverter
